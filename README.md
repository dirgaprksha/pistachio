# Pistachio

A lightweight TypeScript utility library with a clear separation of concerns. Pistachio is a collection of utility packages designed with two distinct purposes:

1. **Core package** - Framework-agnostic, pure utilities for common programming tasks
2. **Adapter packages** - Framework-specific hooks/composables for reactive programming tasks
